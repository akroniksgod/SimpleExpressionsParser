• Для сборки проекта необходима программа Qt | Использовать файл untitled.exe из папки EXE;
• Графический интерфейс реализован как калькулятор для удобного ввода выражения;
• Все кнопки поддерживают ввод с NUMPAD;
• При нажатии кнопки "=" выводится окно с последовательностью действий;
• В классе MainWindow происходит преобразование выражения в обратную польскую нотацию;
• В классе SequenceOfOperations производятся вычисления по обратной польской нотации 
и вывод действий в отдельное окно;
